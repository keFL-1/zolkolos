# Сайт МБУ ЗСЛОО "Золотой колос"

Официальный сайт загородного стационарного лагеря отдыха и оздоровления "Золотой колос" Тимашевского района Краснодарского края.

## Структура проекта

```
├── index.html              # Главная страница
├── about.html              # О нас
├── parents.html            # Родителям
├── children.html           # Детям
├── photos.html             # Фотографии
├── schedule.html           # Распорядок дня
├── shifts.html             # Расписание смен
├── location.html           # Как доехать
├── feedback.html           # Обратная связь
├── reviews.html            # Отзывы
├── staff.html              # Педагогический и вожатский состав
├── contacts.html           # Контакты
├── styles.css              # Основные стили
├── about.css               # Стили для страницы "О нас"
├── schedule.css            # Стили для распорядка дня
└── script.js               # JavaScript функции
```

## Функциональность

### Основные возможности:
- Адаптивный дизайн для всех устройств
- Версия для слабовидящих с настройками:
  - Изменение размера шрифта
  - 5 цветовых схем (белая, черная, синяя, бежевая, зеленая)
  - Управление отображением изображений
  - Синтез речи (озвучивание выделенного текста)
- Интеграция с VK виджетом для новостей
- Чат Chatra для обратной связи
- Яндекс.Карты для отображения местоположения

### Страницы:
1. **Главная** - приветствие и новости из VK
2. **О нас** - полная информация об организации, документы, руководство
3. **Родителям** - необходимые документы и полезная информация
4. **Детям** - информация для детей (заполняется позже)
5. **Фотографии** - фотогалерея (заполняется позже)
6. **Распорядок дня** - подробное расписание дня в лагере
7. **Расписание смен** - даты всех смен
8. **Как доехать** - адрес и карта
9. **Обратная связь** - форма для связи
10. **Отзывы** - отзывы родителей и детей (заполняется позже)
11. **Педагогический состав** - список воспитателей по сменам
12. **Контакты** - телефоны, email, адрес

## Настройка

### Для работы сайта необходимо:

1. **Добавить изображения в папку `images/`:**
   - `temir.jpeg` - фото начальника Темира К.К.
   - `lubomir.webp` - фото заместителя Загадайлова Л.М.
   - `photo1.png` - `photo6.png` - фотографии доступной среды

2. **Настроить Chatra:**
   - Зарегистрироваться на https://chatra.com/ru/
   - Получить ID для домена
   - Заменить `'zolotoj-kolos.ru'` в коде на ваш ID

3. **Настроить VK виджет:**
   - ID группы уже указан: 211788143
   - При необходимости можно изменить параметры отображения

4. **Настроить Яндекс.Карты:**
   - В файле `location.html` вставлена карта
   - При необходимости можно создать свою карту на https://yandex.ru/map-constructor/

## Запуск

Просто откройте `index.html` в браузере. Сайт работает без серверной части.

Для локального сервера можно использовать:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```
